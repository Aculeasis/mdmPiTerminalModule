Для добавления в систему новой колонки сначала надо добавить её в качестве обычного терминала. Это можно сделать в "Панель управления" -> "Настройки" -> "Терминалы"

Там надо указать ip адрес вашей колонки , если хотите по ней слушать онлайнрадио то поставте галочку "может проигрывать медиа-контент" и выбирите "Тип плеера" "MPD", "Порт доступа к плееру" "6600"

Саму колонку можно сделать из недорогого одноплатника, напримар orange pi zero. Установыв туда простой скрипт. Сам скрипт и его установка описаны на странице проекта в GitHub https://github.com/devoff/mdmPiTerminal
