Для добавления в систему нового Голосового ассистента сначала надо добавить его в качестве обычного терминала. Это можно сделать в "Панель управления" -> "Настройки" -> "Терминалы"

Там надо указать ip адрес вашего Голосового ассистента, если хотите по ней слушать онлайн радио то поставьте галочку "может проигрывать медиа-контент" и выберите "Тип плеера" "MPD", "Порт доступа к плееру" "6600"

Самого Голосового ассистента можно сделать из недорогого одноплатника, например, Orange Pi Zero. Установив на него простой скрипт. Сам скрипт и его установка описаны на странице проекта в GitHub https://github.com/devoff/mdmPiTerminal
